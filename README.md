# Vuesax Plugin

## Getting Started
First, install a new projeto with Vue CLI 3, if you don't have one.
```sh
 $ vue create my-new-project
```
After this, added a Vuesax Plugin inside your project
```sh
$ vue add @brun00\vuesax
```
